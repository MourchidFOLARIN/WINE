# WINE — Work IN Excellence 🍷
> **Le Work OS unifié pour les équipes digitales et startups d'Afrique francophone.**

WINE est le projet phare d'**Excellence Team**, un collectif de 6 étudiants en informatique et télécoms de **Lokossa, Bénin**. Après avoir livré avec succès 4 projets (Le TWIN, Academix, Fieri Research, La Nuit du Cœur), l'équipe conçoit WINE : un espace de travail collaboratif ultra-rapide, économique et adapté aux réalités de connectivité et de paiement du continent.

---

## 🚀 Fonctionnalités Clés
WINE propose **6 modules natifs** ou connectés (Notion, Trello, Slack, Google Workspace) pour unifier votre flux de travail :

1. **📁 Gestion de projets :** Tableaux Kanban interactifs avec synchronisation bidirectionnelle tierce.
2. **✓ Suivi des tâches :** Sous-tâches, fils de discussions internes et assignations par membre.
3. **💬 Collaboration Live :** Messagerie instantanée d'équipe avec salons de discussions thématiques.
4. **📱 Planification réseaux :** Calendrier centralisé pour rédiger et planifier vos publications (LinkedIn, Facebook).
5. **📊 Analytique intégrée :** Visualisation de la productivité d'équipe et des métriques connectées.
6. **💼 RH & Business :** Pipeline commercial (CRM d'agence) et gestion des fiches de talents internes.

---

## 🛠️ Stack Technique

- **Frontend :** React (Vite) · CSS3 Moderne · Lucide Icons
- **Backend :** FastAPI (Python) · PostgreSQL · Socket.io (Temps Réel)
- **Sécurité :** Authentification robuste par tokens cryptés JWT

## 💻 Démarrage Local

### Prérequis
- [Node.js](https://nodejs.org/) (version 18+)

### Installation & Lancement
1. Clonez ou ouvrez le dossier du projet :
   ```bash
   cd lendingPages
   ```
2. Installez les dépendances requises :
   ```bash
   npm install
   ```
3. Lancez le serveur de développement local :
   ```bash
   npm run dev
   ```
4. Ouvrez votre navigateur sur [http://localhost:5173/](http://localhost:5173/) pour visualiser la Landing Page interactive.

### Build de Production
Pour compiler et optimiser la Landing Page pour le déploiement de production :
```bash
npm run build
```
